Step 1: Install Python 3.9.13 (if not installed)
Make sure you have Python 3.9.13 installed.

Link:
https://www.python.org/downloads/release/python-3913/

During installation:
*Add Python to PATH
*Install pip

Step 2: Install CMake and Visual C++ Build Tools
Install CMake
Download from:
https://cmake.org/download/

Make sure to add CMake to system PATH during installation.

Install Visual C++ Build Tools
Download from:
https://visualstudio.microsoft.com/visual-cpp-build-tools/

During installation, select:
C++ build tools
Windows 10/11 SDK
MSVC v142 or v143 compiler

Step 3: Install Required Python Packages
Use the following requirements.txt content (or install one by one as shown):

Save this as requirements.txt:

absl-py==2.1.0
attrs==24.2.0
contourpy==1.3.0
cycler==0.12.1
flatbuffers==24.3.25
fonttools==4.54.1
importlib_resources==6.4.5
kiwisolver==1.4.7
matplotlib==3.9.2
numpy==1.24.3
opencv-contrib-python==4.10.0.84
opencv-python==4.7.0.72
opencv-python-headless==4.5.5.64
packaging==24.1
Pillow==9.0.1
protobuf==3.20.3
pyparsing==3.1.4
python-dateutil==2.9.0.post0
six==1.16.0
zipp==3.20.2
Then install:


pip install -r requirements.txt
Step 4: Install dlib (last, after dependencies)

pip install dlib==19.24.6
If you face issues, try:
pip install --upgrade pip setuptools wheel
pip install cmake
pip install dlib==19.24.6
